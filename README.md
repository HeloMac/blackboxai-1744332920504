# Do Zero ao Close - Curso de Maquiagem Profissional

## Project Overview
"Do Zero ao Close" is a comprehensive online course designed for aspiring makeup artists, ranging from beginners to advanced practitioners. The course offers high-quality video lessons, live sessions, and a wealth of resources to equip students with the professional skills needed to thrive in the beauty industry. Led by Heloisa Maceira, a makeup professional, this course empowers students to transform their passion into a successful career by teaching essential techniques, business strategies, and providing ongoing mentorship.

## Installation
This project can be run directly in a web browser without any additional installation. Simply open `index.html` in your preferred web browser to view the course offering.

## Usage
1. Open the `index.html` file in a web browser.
2. Navigate through the sections using the top menu or scroll down to explore:
   - Course Overview
   - Modules
   - Mentoring Options
   - Information about Heloisa
   - Frequently Asked Questions

To register for the course or mentorship program, click the respective buttons that redirect to the payment page.

## Features
- Fully responsive design for both mobile and desktop users.
- Embedded video lessons and live sessions.
- Access to a variety of learning materials (PDFs, checklists, etc.).
- Mentorship program for personalized guidance and support.
- Certification upon course completion.
- Easy navigation with smooth scrolling effects and active menu transitions.

## Dependencies
The project uses the following external dependencies:
- **Tailwind CSS**: For utility-first CSS styling.
- **Font Awesome**: For icon purposes and enhancing UI components.
- **Google Fonts**: For custom typography.

No specific dependencies are listed in a package.json file as the project is primarily an HTML/CSS application.
